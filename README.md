## 能量

想要运气好，一定要记住：

1.不管住哪，要说房子好

2.和谁在一起，就说谁好

3.上班的时候，就说这个工作好

4.遇到晴天，就说阳光很暖

5.碰到下雨天，就说空气很好

6.喝水的时候，就说水都是甜的

人本身就是一个能量场，从你嘴里说出什么，就吸引什么，坚持这样做，你会发现自己一切都越来越好！
